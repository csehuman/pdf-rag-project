### 온브리즈® 브리즈헬러 사용방법

온브리즈® 브리즈헬러는 1일 1회 사용합니다

1.  브리즈헬러의 뚜껑을 위로 당겨 분리한다.
    Figure: 브리즈헬러의 뚜껑을 위로 당겨 분리한다.
    Description: 브리즈헬러의 뚜껑을 위로 당겨 분리한다.

2.  브리즈헬러 열기
    Figure: 브리즈헬러 열기
    Description: 브리즈헬러를 단단히 잡고 흡입구를 젖혀서 브리즈헬러를 연다.

3.  캡슐 준비
    Figure: 캡슐 준비
    Description: 흡입하기 직전에 마른 손으로 캡슐을 블리스터에서 꺼낸다.

4.  캡슐 넣기
    Figure: 캡슐 넣기
    Description: 캡슐을 브리즈헬러의 캡슐 챔버에 넣는다. 캡슐을 흡입구에 직접 넣어서는 안 된다.

5.  브리즈헬러 닫기
    Figure: 브리즈헬러 닫기
    Description: "딱" 소리가 나도록 브리즈헬러를 닫는다.

6.  캡슐 뚫기
    Figure: 캡슐 뚫기
    Description: 브흡입구가 위를 향하도록 브리즈헬러를 잡고, 양쪽의 버튼을 동시에 완전히 눌러 캡슐을 뚫는다. 이 동작은 단 한번만 한다. 캡슐이 뚫릴 때 "딱" 소리가 나야 한다.
